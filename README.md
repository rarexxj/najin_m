# 运行项目
<p>clone项目到本地，进入项目文件夹，安装依赖</p>

```
git clone

git@github.com:rarexxj/najin_m.git
```

```
cd najin_m

npm install

```

然后运行项目

```
npm run dev
```

接着运行 node,启动服务

//启动成功后，将看到输出
// HTTP Server is running in http://localhost:8080
最后打开浏览器，输入localhost:8080即可访问。效果图如下：
